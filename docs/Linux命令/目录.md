## 最小必会命令

下面是命令名称清单，不是可直接粘贴的脚本。涉及删除、提权或结束进程时，先回到相应课程核对目标、权限和退出状态。

```text
pwd
ls -lah
cd
mkdir
touch
cp
mv
rm -- 文件名
cat
less
tail -f
grep
find
chmod
ps
kill -TERM 受控进程PID
df -h
tar
tmux
```
