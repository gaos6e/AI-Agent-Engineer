## 进阶方向（非优先）

| 方向 | 学习点 | 何时学习 |
| --- | --- | --- |
| 模型适配 | Fine-tuning、LoRA、QLoRA | RAG、提示词和工具设计仍无法满足稳定质量要求时。 |
| 开源模型与推理 | 模型蒸馏、模型量化、开源模型部署、推理加速、GPU基础、CUDA基础 | 有私有部署、成本优化或性能约束时。 |
| 对齐与强化学习 | 强化学习、RLHF、RLAIF、对齐技术 | 需要研究模型训练、偏好优化或更深的模型行为控制时。 |
| 前沿系统研究 | AGI系统架构 | 作为研究视野保留，不作为构建 Agent 应用的前置条件。 |
