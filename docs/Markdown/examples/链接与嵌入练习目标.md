# 链接与嵌入练习目标

这是一份课程内的真实目标文件，用于练习笔记链接、标题链接、块链接和 embed。它不包含外部数据或隐藏行为。

## 可观察结果

链接只提供导航；嵌入会在来源笔记中显示目标的当前内容。修改目标后，嵌入显示随之更新，但不会生成第二份副本。

只有输入满足格式要求且人工确认后，流程才可继续。 ^validation-gate
