# 线性代数：坐标、点积、范数与相似度

## 本节目标

向量不是一串没有语义的数字。它通常表示同一对象在若干坐标轴上的分量，例如一条样本的特征、一次调用的质量指标，或一个文本 Embedding。学完本节，你应能区分坐标与对象，手算点积、长度、距离和余弦相似度，并知道这些量何时会误导。

## 坐标依赖约定

二维向量 $x=(3,4)$ 表示“第一坐标为 3、第二坐标为 4”。要让它有工程意义，还要知道：

- 每一维代表什么；
- 单位是什么；
- 维度顺序是否一致；
- 是否做过中心化、缩放或归一化；
- 缺失值和异常值如何处理。

把“延迟毫秒”和“成功率”直接放在同一向量里计算距离时，数值较大的延迟维可能主导结果。线性代数公式不会自动修复单位或数据语义。

## 向量加法与标量缩放

同维向量按坐标相加：

$$
(1,2)+(3,-1)=(4,1)
$$

标量缩放把每个坐标乘同一个数：

$$
2(1,2)=(2,4)
$$

线性组合 $a u+b v$ 就是“先缩放，再相加”。后续的列空间、线性层和低秩分解都建立在这个操作上。

## 点积：对齐程度与加权和

两个同维实向量的点积：

$$
x^Ty=x\cdot y=\sum_{i=1}^{d}x_i y_i
$$

例如 $(1,2)\cdot(3,4)=1\times3+2\times4=11$。点积有两种互补直觉：

1. **代数视角**：把对应分量相乘后求和，是线性模型的加权和。
2. **几何视角**：$x\cdot y=\|x\|_2\|y\|_2\cos\theta$，同时受长度和夹角影响。

点积为 0 的非零向量正交。在二维平面里可理解为垂直；在高维空间里仍表示没有沿彼此方向的投影成分。

## 范数、距离与归一化

欧氏长度（$L_2$ 范数）：

$$
\|x\|_2=\sqrt{\sum_i x_i^2}
$$

两点的欧氏距离是差向量的长度：

$$
d(x,y)=\|x-y\|_2
$$

非零向量可归一化为单位向量：

$$
\hat x=\frac{x}{\|x\|_2}
$$

归一化保留方向、去掉长度。零向量长度为 0，不能除以自身长度；工程实现必须明确拒绝或单独处理。

还有 $L_1$ 范数 $\|x\|_1=\sum_i|x_i|$ 和最大范数 $\|x\|_\infty=\max_i|x_i|$。不同范数定义不同的“近”，会改变正则化、最近邻和鲁棒性语义。

## 余弦相似度

对两个非零向量：

$$
\operatorname{cosine}(x,y)=\frac{x\cdot y}{\|x\|_2\|y\|_2}
$$

它在实向量空间中位于 $[-1,1]$：1 表示同方向，0 表示正交，-1 表示反方向。若向量都已做 $L_2$ 归一化，余弦相似度就等于点积，且平方欧氏距离满足：

$$
\|\hat x-\hat y\|_2^2=2-2\hat x\cdot\hat y
$$

但这不表示余弦分数是“相关概率”。Embedding 模型、文本域、语言和预处理一变，分数分布与可用阈值都可能变化；阈值必须用代表性数据评测。

## 可运行的标准库示例

```python
from math import fsum, sqrt
from typing import Sequence


def dot(left: Sequence[float], right: Sequence[float]) -> float:
    if len(left) != len(right) or not left:
        raise ValueError("vectors must be non-empty and have equal length")
    return fsum(a * b for a, b in zip(left, right))


def norm(vector: Sequence[float]) -> float:
    return sqrt(dot(vector, vector))


def cosine(left: Sequence[float], right: Sequence[float]) -> float:
    denominator = norm(left) * norm(right)
    if denominator == 0.0:
        raise ValueError("cosine is undefined for a zero vector")
    return dot(left, right) / denominator


x = (3.0, 4.0)
y = (4.0, -3.0)
assert dot(x, y) == 0.0
assert norm(x) == 5.0
assert abs(cosine(x, y)) < 1e-12
assert abs(norm((x[0] - y[0], x[1] - y[1])) - sqrt(50.0)) < 1e-12
```

这个示例用于验证公式，不是高性能向量库。生产代码还要检查非有限数、数据类型、批量形状和数值精度。

## 在 Agent 工程中的用途

- 线性模型把特征向量与权重做点积得到得分。
- Embedding 检索常比较归一化向量的点积或余弦相似度。
- Attention 用 query 与 key 的点积形成相对打分，再经过缩放和 softmax；点积本身不是最终概率。
- 梯度、残差和参数更新都可视作高维向量。
- 多指标评测中的“距离”只有在单位、缩放和权重有明确含义时才可靠。

## 常见错误

- 对不同维度或不同坐标语义的向量计算点积。
- 把余弦相似度直接解释为正确率、置信度或因果关系。
- 忘记处理零向量、`NaN` 和无穷值。
- 归一化后仍声称保留了原向量长度信息。
- 在高维空间只凭直觉设置相似度阈值，不做目标数据评测。
