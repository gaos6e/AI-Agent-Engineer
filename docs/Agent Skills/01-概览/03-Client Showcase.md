# Agent Skills 客户端展示

本页列出支持 Agent Skills 格式的 agent 产品和开发工具。原站点使用 JSX 组件展示 logo 和卡片；这里保留产品名、链接和说明，并转换为 Obsidian 友好的普通列表。

> [!info] 文档索引
> 完整页面索引见 <https://agentskills.io/llms.txt>。本页来源为 <https://agentskills.io/clients.md>。

## 客户端列表

- [Junie](https://junie.jetbrains.com/)：Junie 是面向真实开发工作的、与具体 LLM 无关的编码 agent。它基于 IntelliJ Platform 构建，因此能像编辑器一样理解你的项目。
- [Gemini CLI](https://geminicli.com)：Gemini CLI 是开源 AI agent，把 Gemini 的能力直接带到终端中。
- [Autohand Code CLI](https://autohand.ai/)：Autohand Code CLI 是运行在终端中的自主式 LLM 编码 agent。它使用 ReAct（Reason + Act）模式理解代码库、规划变更，并在获得批准后执行。
- [OpenCode](https://opencode.ai/)：OpenCode 是开源 agent，可在终端、IDE 或桌面环境中帮助编写代码。
- [OpenHands](https://openhands.dev/)：OpenHands 是面向云端编码 agent 的开放平台，可从一个 agent 扩展到上千个 agent；它开源、模型无关，并面向企业场景。
- [Mux](https://mux.coder.com/)：Mux 让你能直接在浏览器或桌面中运行多个并行编码 agent，每个 agent 都有独立工作区。Mux 开源，并且不绑定特定 LLM 提供商。
- [Cursor](https://cursor.com/)：Cursor 是 AI 编辑器和编码 agent，可用于理解代码库、规划并构建功能、修复 bug、审查变更，并与已有工具协作。
- [Amp](https://ampcode.com/)：Amp 是面向前沿模型的编码 agent，让用户直接调用主流强模型的完整能力。
- [Letta](https://www.letta.com/)：Letta 是构建有状态 agent 的平台，支持带高级记忆、可学习并持续自我改进的 AI。
- [Firebender](https://firebender.com/)：Firebender 是 Android 原生编码 agent，可编写功能、在模拟器中测试，并自动修复问题。
- [Goose](https://block.github.io/goose/)：Goose 是开源、可扩展的 AI agent，不止提供代码建议，还可安装、执行、编辑和测试，并可搭配任意 LLM。
- [GitHub Copilot](https://github.com/)：GitHub Copilot 直接在编辑器中协同工作，提供整行代码或完整函数建议。
- [VS Code](https://code.visualstudio.com/)：Visual Studio Code 兼具代码编辑器的简洁性和开发者在编辑、构建、调试循环中需要的核心能力。
- [Claude Code](https://claude.ai/code)：Claude Code 是 agentic coding 工具，可读取代码库、编辑文件、运行命令，并与开发工具集成；支持终端、IDE、桌面应用和浏览器。
- [Claude](https://claude.ai/)：Claude 是 Anthropic 面向问题解决者的 AI，可处理复杂挑战、分析数据、编写代码，并辅助推理高难工作。
- [OpenAI Codex](https://developers.openai.com/codex)：Codex 是 OpenAI 面向软件开发的编码 agent。
- [Piebald](https://piebald.ai)：Piebald 是桌面和 Web 应用，让 agentic development 更容易，同时让用户完全控制配置、上下文和流程。
- [Factory](https://factory.ai/)：Factory 是 AI 原生软件开发平台，覆盖从 IDE 到 CI/CD 的工作场景。用户可以把重构、事故响应、迁移等完整任务委派给 Droids，而无需改变工具、模型或工作流。
- [pi](https://shittycodingagent.ai/)：Pi 是一个极简终端编码 harness，强调让工具适配用户工作流。
- [Databricks Genie Code](https://databricks.com/)：Genie Code 是 Databricks 中面向数据工作的自主式 AI 伙伴。
- [Agentman](https://agentman.ai/)：Agentman 是 agentic 医疗健康平台，用 AI agent 自动化收入周期工作流，同时保留控制权；每个动作都可测试、可追踪、可审计。
- [TRAE](https://trae.ai/)：Trae 是自适应 AI IDE，通过协作式工作方式提升开发速度。
- [Spring AI](https://docs.spring.io/spring-ai/reference)：Spring AI 旨在简化包含人工智能功能的应用开发，减少不必要的复杂性。
- [Roo Code](https://roocode.com)：Roo Code 把一支 AI 开发团队放进编辑器，提供深度项目级上下文、多步骤 agentic coding 和面向开发者的灵活性。
- [Mistral AI Vibe](https://github.com/mistralai/mistral-vibe)：Mistral Vibe 是由 Mistral 模型驱动的命令行编码助手，提供与代码库对话的界面，让用户用自然语言探索、修改并操作项目。
- [Command Code](https://commandcode.ai/)：Command Code 是会持续学习用户编码偏好的编码 agent。其 `taste-1` 元神经符号模型结合 LLM 与用户编码品味，并通过持续强化学习改进。
- [Ona](https://ona.com)：Ona 是后台 agent 平台，可在云端运行一组 AI 软件工程师，并在内核层进行编排、治理和安全控制。
- [VT Code](https://github.com/vinhnx/vtcode)：VT Code 是开源编码 agent，具备 LLM 原生代码理解和稳健的 shell 安全机制，支持多个 LLM 提供商、自动故障转移和高效上下文管理。
- [Qodo](https://www.qodo.ai/)：Qodo 是 agentic 代码完整性平台，将 AI 融入审查、测试和编写流程，以在开发各阶段强化代码质量。
- [Laravel Boost](https://github.com/laravel/boost)：Laravel Boost 通过必要指南和 Agent Skills 加速 AI 辅助开发，帮助 AI agent 编写符合 Laravel 最佳实践的高质量 Laravel 应用。
- [Emdash](https://emdash.sh)：Emdash 是不绑定提供商的桌面应用，可并行运行多个编码 agent，每个 agent 隔离在自己的 git worktree 中，并支持本地或通过 SSH 连接远程机器。
- [Snowflake Cortex Code](https://docs.snowflake.com/en/user-guide/cortex-code/cortex-code)：Cortex Code 是集成在 Snowflake 平台中的 AI 驱动智能 agent，面向复杂数据工程、分析、机器学习和 agent 构建任务优化。
- [Kiro](https://kiro.dev/)：Kiro 通过规格驱动开发为 AI 编码带来结构化流程，帮助用户更好地完成工作。
- [Workshop](https://workshop.ai/)：Workshop 是跨平台 AI 编码 agent，用于构建完整应用。它支持多 LLM、子 agent、自定义 agent 和技能，并提供桌面应用、Web 应用和 CLI。
- [Google AI Edge Gallery](https://github.com/google-ai-edge/gallery)：Google AI Edge Gallery 是在移动设备上运行强大开源大语言模型的主要入口。
- [nanobot](https://nanobot.wiki/)：nanobot 是超轻量、开源的个人 AI agent，可运行在终端、Telegram、Discord、Slack、WeChat 等多个平台，并内置 MCP 支持和可扩展技能系统。
- [fast-agent](https://fast-agent.ai/)：fast-agent 是与 LLM 交互的简单、可扩展方式，适合编码、eval、ACPX 和 Skills 开发。
