# Agent Skills

## 知识库简介

Agent Skills 是一种用文件夹、`SKILL.md`、脚本和资源为 AI agent 提供可复用专业能力的开放格式。它把“完成一类任务所需的程序性知识”放进可发现、按需加载、可测试的包中。这里先学习格式、触发、渐进式披露、安全和评测，再把任务开始前已有的 8 篇官方资料作为参考层使用。

> [!info] 来源与范围
> 官方文档索引为 <https://agentskills.io/llms.txt>，规范与创作指南在 2026-07-14 复核。格式是开放的，但安装位置、可用工具、触发实现和权限提示仍取决于具体 client；不要把某个产品行为误写成通用规范。本文把规范事实、官方创作建议和本课程的严格教学约定分开标注。

## 在总路线中的位置

它位于“单 Agent 与工具”阶段。先掌握 [[Agent 核心/00-目录|Agent 核心]] 的任务循环与 [[Tool Calling（含 Function Calling）/00-目录|Tool Calling]]，再把稳定工作流封装为 Skill；需要连接外部工具或数据时，与 [[MCP/00-目录|MCP]] 组合。

## 学习目标

- 解释开放格式与具体 agent client 的边界。
- 创建符合规范的目录与 `SKILL.md` YAML frontmatter。
- 利用 metadata→instructions→resources 三层渐进式披露控制上下文成本。
- 写出既能覆盖真实意图、又不会过度触发的 `description`。
- 设计无交互、安全默认、结构化输出的脚本，并按需组织 references/assets。
- 用格式校验、触发测试、有/无 Skill 基线和断言迭代质量。

## 前置知识

- 能读写 Markdown 和简单 YAML frontmatter。
- 会在 PowerShell 7 运行 Python 3 脚本并查看退出码。
- 理解 agent 的上下文窗口、工具调用和“最小权限”概念。
- 不要求先熟悉某个 client；实际安装前再读该 client 的官方文档。

## 推荐学习顺序

1. [[Agent Skills/学习路线/01-开放格式与目录结构|开放格式与目录结构]]：分清规范核心和 client 自己的实现选择。
2. [[Agent Skills/学习路线/02-SKILL前置元数据与渐进披露|SKILL.md 前置元数据与渐进式披露]]：准确写 frontmatter，并把上下文按需拆分。
3. [[Agent Skills/学习路线/03-触发描述与范围边界|触发描述与范围边界]]：用真实正反例校准 `description`。
4. [[Agent Skills/学习路线/04-脚本资源与安全|脚本、资源与安全]]：设计可执行资源与最小权限边界。
5. [[Agent Skills/学习路线/05-测试评测与迭代|测试、评测与迭代]]：比较“有 Skill”和基线，并以证据改进。
6. [[Agent Skills/学习路线/06-项目-创建并验证技能|项目：创建并验证技能]]：运行一个不联网的完整示例。

## 动手实践或项目入口

- 主项目：[[Agent Skills/学习路线/06-项目-创建并验证技能|创建并验证 text-statistics Skill]]。
- 触发练习：先为你的 Skill 写正反各 8～10 条真实请求，负例优先选择容易误触发的相邻任务；再留出未参与调参的验证集。
- 评测练习：为一个机械可验证结果写断言，并比较不加载 Skill 时的基线。

## 掌握标准

- [ ] 能从零建立含 `SKILL.md` 的合法目录，并解释 name 必须匹配父目录。
- [ ] 能说明为什么启动时只加载 name/description，以及长参考应何时读取。
- [ ] 能写出同时包含“做什么”和“何时使用”的 description，并用近似负例检验范围。
- [ ] 能审查脚本的网络、文件、命令、凭据和输出边界。
- [ ] 能运行本地示例验证器与脚本，理解它们的覆盖范围和局限。
- [ ] 能用有 Skill/无 Skill 基线、具体断言和人工复核共同决定是否改版。

## 与其他知识库的关系

- [[提示词工程/00-目录|提示词工程]] 关注单次或模板化指令；Skill 将稳定流程、脚本和参考资料打包，并通过描述触发。
- [[上下文工程/00-目录|上下文工程]] 解释为何要按需加载；渐进式披露是其具体工程应用。
- [[MCP/00-目录|MCP]] 暴露工具、资源和 prompts；Skill 可以教 agent 何时、如何安全使用这些能力。
- “评测体系”知识库提供更完整的实验设计、指标与误差分析方法；总库完成后可从主路线进入。
