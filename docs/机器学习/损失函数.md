## 交叉熵(Categorical Cross Entropy, CE)

1. 类别总数：$C$
2. 样本总数：$N$
3. 第 $i$ 个样本的真实标签：$y_i \in \{1,2,\dots,C\}$
4. 模型输出的预测概率分布：$\hat{p}_i = (\hat{p}_{i1}, \hat{p}_{i2}, \dots, \hat{p}_{iC})$，其中 $\hat{p}_{ij}$ 表示样本 $i$ 属于类别 $j$ 的预测概率。
5. yij​ 是one-hot 标签，即：

   ```
   $$y_{ij} = \begin{cases}
   1 & \text{若真实类别为 } j \\
   0 & \text{否则}
   \end{cases}$$
   ```

```
$$\mathcal{L}_{CE} = -\frac{1}{N} \sum_{i=1}^N \sum_{c=1}^C y_{i,c} \log(\hat{y}_{i,c})$$
```

## **二元交叉熵 (Binary Cross Entropy, BCE）**

```
$$\mathcal{L}_{BCE} = -\frac{1}{N} \sum_{i=1}^N \Big[ y_i \log(\hat{y}_i) + (1-y_i)\log(1-\hat{y}_i) \Big]$$
```

## 均方误差

1. 第 $i$ 个样本的真实值为 $y_i$
2. 一共有 $N$ 个样本
3. 模型预测值为 $\hat{y}_i$

```
$$\mathcal{L}_{MSE} = \frac{1}{N}\sum_{i=1}^N (y_i - \hat{y}_i)^2$$
```

## **对比损失 (Contrastive Loss)**

```
$$\mathcal{L} = \frac{1}{2N} \sum_{i=1}^N \Big[ y_i \cdot D^2 + (1-y_i) \cdot \max(0, m - D)^2 \Big]$$
```
