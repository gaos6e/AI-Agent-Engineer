# 获取帮助

与 LangChain 社区建立联系，获取学习资源，并获得自信构建所需的支持。

## 学习资源

通过我们全面的学习材料开始您的旅程或加深您的知识。

* **[聊天 LangChain](https://chat.langchain.com/)**：向文档询问有关 LangChain 的任何信息，由实时文档提供支持
* **[API参考](https://reference.langchain.com/python/)**：所有LangChain包的完整文档

## 社区支持

通过我们活跃的社区渠道获得其他开发人员和 L​​angChain 团队的帮助。

* **[社区论坛](https://forum.langchain.com/)**：提出问题、分享解决方案并讨论最佳实践
* **[Community Slack](https://www.langchain.com/join-community)**：与其他构建者联系并获得快速帮助

## 专业支持

对于企业需求和关键应用程序，请访问专用支持渠道。

* **[支持门户](https://support.langchain.com/)**：提交票证并跟踪支持请求
* **[LangSmith status](https://status.smith.langchain.com/)**：LangSmith 服务和 API 的实时状态

## 贡献

帮助我们为大家改进LangChain。无论您是要修复错误、添加功能还是改进文档，我们都欢迎您的贡献。

* **[贡献指南](https://docs.langchain.com/oss/python/contributing/overview)**：关于为LangChain做贡献你需要了解的一切

## 保持联系

关注我们，了解最新更新、公告和社区亮点。

* **[X (Twitter)](https://x.com/langchain)**：每日更新和社区焦点
* **[LinkedIn](https://www.linkedin.com/company/langchain/)**：专业网络和公司更新
