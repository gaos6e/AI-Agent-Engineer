# Link and Embed Practice Target

This is a real target file inside the course for practicing note links, heading links, block links, and embeds. It contains no external data or hidden behavior.

## Observable result

A link only provides navigation; an embed displays the target's current content in the source note. After the target changes, the embed changes too, but no second copy is created.

The flow can continue only after input satisfies format requirements and a human confirms it. ^validation-gate
