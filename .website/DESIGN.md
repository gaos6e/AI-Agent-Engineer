# AI Agent Engineer 网站设计规范

## 品牌定位

- 名称：AI Agent Engineer
- 副标题：从零构建、评测与部署 AI Agent
- 核心感受：严谨、清晰、可操作、有探索感；像一套持续生长的工程课程，而不是营销落地页或霓虹风控制台。
- 自定义视觉方向：**Signal Foundry（信号工坊）**。以精密网格、信息路径和“信号点”表达从知识到工程能力的转化。

## 信息架构

- 首页：品牌说明、全文搜索、8 个阶段、53 个知识库入口、学习方式与内容统计。
- 路线：按 8 个阶段展示 53 个知识库，不出现学习进度、勾选或账号状态。
- 文档页：顶部品牌与搜索、左侧课程导航、正文、右侧页内目录、上下篇与相关链接。
- 移动端：单列正文；导航与目录使用可关闭抽屉；搜索保持一键可达。
- 代码和数据资产：`.py`、`.json`、`.csv`、`.ipynb` 作为只读预览/下载资源，不生成独立文档页面。

## 色彩

主色以深墨蓝为背景，青色只作为可操作信号和当前状态，不做大面积霓虹装饰。

| Token | 色值 | 用途 |
| --- | --- | --- |
| `--ink-950` | `#06101D` | 页面主背景 |
| `--ink-900` | `#0A1727` | 导航与浮层 |
| `--ink-800` | `#10233A` | 卡片、代码区背景 |
| `--paper-100` | `#EAF4FF` | 主文字 |
| `--paper-300` | `#AFC2D6` | 次级文字 |
| `--signal` | `#49D7FF` | 链接、焦点、当前项 |
| `--signal-deep` | `#168DB4` | 边框和交互按下态 |
| `--line` | `rgba(135, 190, 224, 0.18)` | 分隔线和网格 |

约束：不使用渐变文字；不使用紫蓝渐变背景；不把每个模块都做成玻璃卡片；青色高光面积应显著小于正文和背景。

## 字体与排版

- 标题：`Bahnschrift` / `Microsoft YaHei UI` / 系统无衬线，使用 600–700 权重与紧凑字距。
- 正文：`Microsoft YaHei UI` / `PingFang SC` / `Hiragino Sans GB` / 系统无衬线，暗色模式采用较轻字重和更宽行高。
- 数据、阶段编号、路径和代码：`Cascadia Code` / `JetBrains Mono` / `Consolas` / 等宽字体。
- 正文最大行宽约 74 个中文字符；文档正文默认 17px、行高 1.85；首页 Hero 标题使用 `clamp()` 在 44–84px 间响应。
- 通过标题、正文、等宽标签三种“语气”建立层次，不额外引入在线字体依赖。

## 布局语言

- 12 列桌面网格，首页采用非对称布局：主叙事占 7–8 列，路线/统计占剩余列。
- 阶段模块允许不同跨度和信息密度，避免 8 个完全相同的卡片。
- 边线、编号、短横线用于建立阅读路径；背景只保留轻量网格和局部径向光晕。
- 文档页优先阅读：桌面三栏，平板双栏，移动单栏；侧栏不挤压正文至不可读宽度。

## 动效原则

- 先保证无 JavaScript 时最终布局完整可用；GSAP 只做渐进增强。
- 首页首屏采用 0.2 秒延迟后的分层进入：品牌标记、标题、说明、操作区分别使用不同方向和时长；总进入时间不超过 1.2 秒。
- 阶段与知识库使用 `ScrollTrigger.batch()` 做一次性可见区进入，只动画 `transform` 和 `opacity`，单批次总 stagger 不超过 0.5 秒。
- 文档导航采用极短的状态过渡；不使用滚动劫持、长时间 pin、无限动画、重度视差或持续跟随光标。
- SPA 导航前清理所有 tween/ScrollTrigger；导航完成后重新初始化并按需 `refresh()`。
- `prefers-reduced-motion: reduce` 时完全跳过进入和滚动动效，保留即时状态变化。

## 交互与可访问性

- 所有交互均可通过键盘完成；焦点环使用高对比青色且不被阴影替代。
- 正文字体、导航、按钮和代码区域达到 WCAG AA 对比度；正文链接同时依靠颜色和下划线/形状识别。
- 触控目标最小 44×44px；移动抽屉打开时锁定背景滚动并正确恢复焦点。
- 搜索、导航和资源预览拥有明确的可访问名称；资源预览可用 `Escape` 关闭。
- Mermaid、图片和表格在窄屏可横向滚动或缩放，不截断关键内容。

## 明确不做

- 登录、评论、云端同步和学习进度。
- 首页自动播放视频、粒子海洋、WebGL 背景或与学习无关的装饰性 3D。
- 为追求统一而改写源知识库正文。
- 将完整第三方复刻、许可不明附件或 Vault 其他区域带入构建产物。
